Input Files:
1) "t2SOT_rawData.csv" - Spatial Orientation Task raw data. Rows = participants, cols = items on task(1-12), cells = un-corrected angular error.

2) "t2Model_rawData.csv" - Model Building Task raw data. Rows = participants (8 rows per participant), cols = date of test, model number, building name, x (independent coordinate), y (independent coordinate), error.

3) "t2Recall_rawData.csv" - Building Recall Task raw data cleaned. Rows = participants, cols = duration(seconds), Q1-Q8 building recall.

4) "t2MRT_SDTstyle.csv" - Mental Rotation Task, spatial dimensionality test (SDT)-style score. Rows = participants, col = score.

5) "t2Demo.csv" - liscence variable (0= no, 1= yes, 2= permit)
